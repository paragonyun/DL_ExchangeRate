## 📃 파일 설명
`Normal_Seq2Seq` : 성능 비교를 위한 가장 기본적인 형태의 Seq2Seq 모델입니다. 

`Attention_Seq2Seq` : 기본적인 Seq2Seq에 Attention이 적용된 모델입니다.  

`DARNN_Seq2Seq` : Dual stage Attention이 적용된 모델입니다.
